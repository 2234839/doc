# 最小值

- pubdate:2019-07-04 16:14:40
- tags:习题,javascript

---------

题目:

前一章介绍了标准函数Math.min，它可以返回参数中的最小值。我们现在可以构建相似的东西。编写一个函数min，接受两个参数，并返回其最小值。

解:

````javascript
function min(number1,number2){
    return number1 < number2 ? number1 : number2
}

console.log(min(0, 10));
console.log(min(0, -10));
````

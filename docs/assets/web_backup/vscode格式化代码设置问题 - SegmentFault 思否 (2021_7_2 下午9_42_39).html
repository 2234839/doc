你好，我使用 volar 来提供 vue 文件的语法提示等支持，
在使用如下方式初始化项目

❯ npm init @vitejs/app
√ Project name: ... test
√ Select a framework: » vue
√ Select a variant: » vue-ts

然后在项目的根目录下创建 .prettierrc 文件
内容为：

{
    "singleQuote": true,
    "semi": false,
    "trailingComma": "none"
}

再对 vue 文件进行格式化即可实现你所提出的四点要求

typescript 与 vuex，提供代码提示

pubdate:2019-11-10 19:50:04
tags:typescript,vuex
vuex commit 是没有代码提示的，当初接触到 typescript 之后就一直想实现这个提示，现在终于做到了
下面是实现方式
typescript

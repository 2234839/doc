github releases 加速下载

在github的releases页面选择需要下载的文件，右键复制需要下载文件的链接地址。
新建一个浏览器页面，将链接地址粘贴到浏览器地址栏，将github.com改成 hub.fastgit.org， 回车下载速度起飞。
或者在github.com后面加上.cnpmjs.org

通过ssh在远程服务器上执行命令

要在远程服务器上执行命令而不让当前终端完全 SSH 连接到远程终端，可以使用 SSH 的 "ssh remotehost command" 语法。这种方式会在远程主机上执行指定的命令，然后返回结果。
以下是如何使用这种方式来执行远程命令的示例：
bash
ssh user@remotehost 'command'
​user​ 是你在远程主机上的用户名。
​remotehost​ 是远程主机的 IP 地址或域名。
​'command'​ 是你想要在远程主机上执行的命令，用引号括起来。
举个例子，如果你想要在远程主机上执行 ls -l​ 命令，并返回结果，可以使用以下命令：
bash
ssh user@remotehost 'ls -l'
当你运行这个命令时，它会通过 SSH 连接到远程主机，执行指定的命令，然后将结果输出到当前终端。

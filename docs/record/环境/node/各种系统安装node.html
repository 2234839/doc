apt 安装 node

要使用 apt 安装新版本的 Node.js，您需要遵循以下步骤：
1.
首先，更新您的包列表：
plaintext
sudo apt update
2.
安装允许 apt 通过 HTTPS 使用存储库的必要包：
plaintext
sudo apt install curl gnupg2 ca-certificates lsb-release
3.
添加 NodeSource 存储库的 GPG 密钥：
plaintext
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
这个命令会添加 Node.js 的长期支持（LTS）版本。如果您想安装最新的稳定版本，可以将 setup_lts.x​ 替换为 setup_current.x​。
4.
现在，您可以安装 Node.js：
plaintext
sudo apt install nodejs
5.
安装完成后，验证安装：
plaintext
node --version npm --version
6.
（可选）如果您需要编译原生插件，还需要安装 build-essential 包：
plaintext
sudo apt install build-essential
注意事项：
这个方法会安装 Node.js 和 npm（Node 包管理器）。
使用 NodeSource 存储库可以确保您获得最新的 Node.js 版本。
如果您之前已经安装了旧版本的 Node.js，这个过程会更新到最新版本。
请确保您有足够的权限执行这些命令（使用 sudo）。
如果您在安装过程中遇到任何问题，或者需要更详细的解释，请随时告诉我。我很乐意提供进一步的帮助。

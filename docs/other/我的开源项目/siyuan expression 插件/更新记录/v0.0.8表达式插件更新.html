插件会不断对发生变化的块进行求值，但为了避免产生大量运算和读写，依据块的 update 字段进行了一些优化，在插件初次启动时会对所有表达式进行一次求值，之后只会对更新了的块的表达式进行求值。
如果想要触发求值，可以手动随意修改一下块内容。

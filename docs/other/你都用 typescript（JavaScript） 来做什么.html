所有能用他做的东西_(:з」∠)_。
用 js 是因为 js 真的对新人很友好，因为可以在任何一个浏览器运行，马上就可以看到效果。
用 ts 是因为 js 难以重构，在缺乏 ts 的时代连像样的智能提示都没有。
现在其实可以写纯粹的 js 然后利用 jsdoc 与 d.ts 补上类型，我是在 svelete/kit 中明白这种用法优势的。

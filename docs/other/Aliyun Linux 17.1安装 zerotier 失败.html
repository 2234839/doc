Aliyun Linux 17.1安装 zerotier 失败
分析
主要错误是 http://download.zerotier.com/redhat/el/17.01/repodata/repomd.xml: [Errno 14] HTTP Error 404 - Not Found 具体报错信息
通过查看 http://download.zerotier.com/redhat/el/ 可知应该是 curl -s https://install.zerotier.com | sudo bash 命令内依赖了系统版本这个数据，但由于我使用的是 Aliyun Linux 17.1 所以他找不到相关的信息
解决
手动下载 https://install.zerotier.com 这里的脚本，
这里有一个 $releaserver 基本验证了 这里的推测
我改成下图这样后再手动执行脚本安装成功了

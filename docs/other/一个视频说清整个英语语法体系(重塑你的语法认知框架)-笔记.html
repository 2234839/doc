我们为什要学语法？
为了造句
长语句由基本句组成
基本句就是不能再拆的句子
简单句，Simple Sentences
简单句粗略的分为什么怎么样
什么（主语）与怎么样（谓语）就是 句子成分
什么（人或物）+ 怎么样（『 动作 』：发生了什么事情）

并非一般意义上动作，更接近 『 动词』
有哪些 动作 ？
可以独立完成的动作：不及物动词
Papa Rabbit sleeps => 主语 +（不及物）动词
兔老爹睡觉
有一个动作的承受者：及物动词
Papa Rabbit likes you => 主语 +（单及物）动词 + 宾语（动作的承受者）
兔老爹喜欢你
有两个动作的承受者：双及物动词
Papa Rabbit teaches you English=> 主语 +（双及物）动词 + 间接宾语（动作的承受者）+直接宾语
兔老爹教 你 英语
只有一个动作承受者但不同于 有一个动作的承受者：及物动词 ：复杂及物动词
Papa Rabbit considers you smart=> 主语 + 复杂及物动词 + 宾语 + （宾语）补语
兔老爹 认为 你 聪明
这里的聪明作为补充信息就是（宾语）补语
（不是个『 动作』）
Papa Rabbit is tall. => 主语 （主语）补语
中文中 is 经常翻译为「是」但他实际上是将 tall 赋予给前面的宾语
以 is 为代表的一类词的作用就是把这个词后面的信息赋予给前面的词。(连)系动词
is => Rabbit <- tall

continue

这是一个非常棒的开源ai代码助手

添加自动完成

在 config.json 中添加如下配置，在我的电脑上（RTX 2060）qwen2.5-coder:1.5b 有不错的自动补全速度，deepseek-coder:6.7b 勉强能用
js
"tabAutocompleteModel": { "title": "Tab Autocomplete Model", "provider": "ollama", "model": "qwen2.5-coder:1.5b" },

自定义斜杠命令

在 config.json 中定义
js
"customCommands": [ { "name": "test-customCommands", "prompt": "{{{ input }}}\n\n使用中文回答我的提问，在回答的最后提示你的知识最新更新时间", "description": "This is an example custom command. Open config.json to edit it and create more" } ],
或者参考 https://docs.continue.dev/actions/how-to-customize#custom-slash-commands-with-configts 实现以编程方式定义

添加 prompt

在项目文件夹下添加如下文件
.continue/prompts/api创建文档生成表单代码.prompt

添加全局提示词

编写在如下文件中
.continuerules

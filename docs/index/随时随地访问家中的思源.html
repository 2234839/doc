随时随地访问家中的思源

昨天在使用 vscode tunnel 时发现了这个神奇的技巧
只需要一个vscode加账号，家里的电脑开机，其它地方随时可访问 甚至客户端都不用安装vscode他支持浏览器打开的 大致就是家里的电脑安装vscode
vscode中点击账号图标登录账号开启远程隧道访问
复制链接浏览器中打开然后在开启6806端口映射

🔧 准备工作

1.
家中电脑保持开机状态
2.
安装最新版 VS Code（≥1.77 版本）
3.
拥有 Microsoft/GitHub 账号

🚀 核心操作步骤

1.
安装 Remote - Tunnels 扩展
2.
点击左下角「远程连接」图标
​ ​
3.
选择「Connect to Tunnel...」
4.
登录 Microsoft/GitHub 账号
5.
开启隧道时会生成两类链接：
Web 版：https://vscode.dev/tunnel/<设备名>/<路径>
桌面版：vscode://vscode-remote/tunnel/<设备名>/<路径>

🔌 端口映射技巧（以6806为例）

​ ​
1.
打开之前vscode提供的链接，或者使用vscode 客户端连接远程pc
2.
在下面的端口面板添加6806端口映射
3.
然后就可以得到一个链接可以在任意浏览器打开了
4.
安全方面有微软的账号机制保证

🌐 访问方式

✅ 浏览器直接访问（无需安装任何软件）

超级棒的remote app

qq 微信... 这些软件实在是难以避免使用，但我又不想再被这些个流氓软件扫盘了，于是我使用一台电脑用 hyper-v 开启一个专门供这些流氓玩耍的 win10 虚拟机，再利用 remote app 在主力电脑上面使用。
远程运行的微信：
​ ​
可以将其他电脑运行的程序通过 rdp 的形式在本地打开，这样既有了 rdp 的各种方便复制的特性，又不用那么麻烦的管理窗口堆栈（不用 remote app 之前首先需要找到对应的 rdp 窗口，然后再找到对应的软件，切换特别麻烦）。
我很早就听说的这个功能，只是现在才开始尝试，非常惊艳，不过还是需要有一个不错的网络环境才行的。

使用方式

kimmknight/remoteapptool 在实际运行远程 app 的电脑上安装这个软件，然后生成对应的 rdp 文件，生成的时候要填写正确的 ip 地址。
然后将 rdp 文件复制到你使用远程app的电脑上面，就可以点击打开运行了，局域网内体验非常棒。

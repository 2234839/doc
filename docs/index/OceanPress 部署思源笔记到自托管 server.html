OceanPress 部署思源笔记到自托管 server

​ ​
大致流程：
1.
安装 nodejs 和 pnpm / npm
2.
安装 OceanPress server 程序 : pnpm i -g oceanpress-server​
3.
配置 apiKey ： 在运行程序的目录中创建文件 .env 并写入配置 API_KEY=api-key-test ,请注意不要泄露 apiKey，另外创建复杂度高一些的
4.
运行 OceanPress server 程序 : 命令行执行 oceanpress_server​
5.
OceanPress 中配置 OceanPress server apiBase 和 ApiKey

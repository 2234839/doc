自定义配置

rime输入法较为极客，他的ui界面基本上配置不了什么东西。
需要修改对应的文件才能进行配置（以 ubuntu 为例该配置文件位于：/home/用户名/.config/ibus/rime/default.custom.yaml）
下面是我的配置，可以看出来这个 custom.yaml 文件就是简单的 patch 覆盖 default.yaml 中的选项
我主要定制了如下
从rime默认的繁体配置为简体
采用双拼（需要另外的命令安装双拼功能）
屏蔽 ctrl​ + `​ （grave）快捷键（和 vscode 冲突了）
yaml
patch: schema_list: - schema: luna_pinyin_simp # 简化字模式 - schema: double_pinyin_flypy # 小鹤双拼 switcher: caption: 〔方案選單〕 hotkeys: # - Control+grave #grave = ` - Control+Shift+grave - F4 save_options: - full_shape - ascii_punct - simplification - extended_charset - zh_hant - zh_hans - zh_hant_tw fold_options: true abbreviate_options: true option_list_separator: '／'
然后点击状态栏的 rime 图标点击部署即可。 ​

配置双拼

首先在开始菜单中打开“【小狼毫】輸入法設定”
点击“獲取更多輸入方案...”，在打开的窗口中输入double-pinyin然后回车
输出中看到Everything is OK字样则表示安装成功

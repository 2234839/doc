## 修改属性后没有出发视图或者计算属性等更新

这个多半是因为你修改的对象原本没有这个属性，因为目前 vue 使用 Object.defineProperty 实现数据的响应式，而这个方法不可能监听到任意属性的修改，所以需要先使用 Vue.set 将相关属性设置为响应式。 也许以后 vue 使用 proxy 实现的时候就不用管这个问题了

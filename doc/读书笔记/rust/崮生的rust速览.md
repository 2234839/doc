## rust 项目结构
{: id="20201227174018-vw6sy1c"}

```
hello-rust
|- Cargo.toml
|- src
  |- main.rs
```
{: id="20201227175010-0i6gs1q"}

`Cargo.toml` 为 Rust 的清单文件。其中包含了项目的元数据和依赖库。
{: id="20201227175011-bg2a5yb"}

`src/main.rs` 为编写应用代码的地方。
{: id="20201227175011-2kd0awj"}
